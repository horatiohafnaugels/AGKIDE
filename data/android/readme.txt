This folder should contain the following files for all platforms
 -a lib folder with the AGK interpreter libs
 -a sourceGoogle folder with the raw AndroidManifest.xml, resMerged, resOrig, and the compiled classes.dex
 -a sourceAmazon folder with the raw AndroidManifest.xml, resMerged, resOrig, and the compiled classes.dex
 -a sourceOuya folder with the raw AndroidManifest.xml, resMerged, resOrig, and the compiled classes.dex
 -assets folder containing nothing
 -android31.jar
 -apksigner.jar
 -bundletool.jar
 -debug.keystore
And the platform specific version of the following files
 -aapt2-bundle.exe
 -libwinpthread-1.dll (Windows only)
 -zipalign.exe
